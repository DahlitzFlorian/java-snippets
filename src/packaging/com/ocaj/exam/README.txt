How to compile and run the files.
==================================

Go to the root directory and type:

javac -d bin -cp .\src src\packaging\com\ocaj\exam\GreetingUniverse.java
javac -d bin -cp .\src src\packaging\com\ocaj\exam\planets\*.java

To run it type:

java -cp .\bin packaging.com.ocaj.exam.GreetingUniverse

Done!
